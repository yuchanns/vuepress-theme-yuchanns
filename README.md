## Fetch
```sh
yarn add vuepress-theme-yuchanns # or npm install vuepress-theme-yuchanns
```

## Dev
Feel free to fork && pull request!
```sh
git clone git@github.com:yuchanns/vuepress-theme-yuchanns.git
cd vuepress-theme-yuchanns
yarn
yarn dev
```
